MEANP-CLI
================

A simple command line tool for creating front-end modules on the meanp-seed project 
 
### Setup:
```
npm install -g meanp
```

### Commands:
```yo meanp <name>```
Clones the meanp-seed project to a new directory with the name you enter.

```yo meanp:modules <name>```
